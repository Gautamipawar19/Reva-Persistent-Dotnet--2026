# Fleet Manager 🚗📦

A simple C# console application to manage a vehicle fleet.

## Features
- Abstract Vehicle base class
- Car and Truck implementations
- Interfaces for Maintenance and Tracking
- Polymorphism using List<Vehicle>
- Encapsulated validation
- Unit testing support

## Technologies Used
- .NET 8
- C#
- Console Application
- xUnit (for testing)

## Project Structure
FleetManager
- Vehicles
- Interfaces
- Helpers
- Program.cs

## Known Limitations
- No database
- Console-based UI only
